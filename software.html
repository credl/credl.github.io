Software

I was involved in the following list of software projects.

dlvhex (Reasoner for HEX-Programs)


A logic-programming reasoner for evaluating HEX-progams, which are an extension of answer set programs.

MELD (Belief Merging System)


System for merging of declarative knowledge bases.

Angry-HEX (AI agent for AngryBirds)


An AI agent for the Angry Birds computer game, based on declarative problem solving with logic programs.

ABC Benchmarking System (automated benchmarking system)


A scripting system for automatic and parallel benchmarking and generation of result tables in LaTeX format based on HTCondor and R.

Clingo App (port of clingo as app for mobile platforms Android and iOS)


A port of the clingo reasoner to Android-based systems (experimental project).
